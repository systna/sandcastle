## What is Sandcastle?

Sandcastle is a TypeScript CLI tool that orchestrates AI coding agents inside isolated sandbox environments. It manages the full lifecycle of running an agent against your codebase: syncing code into a sandbox, invoking the agent, and syncing changes back.

## Key Concepts

- **Sandbox** -- An isolated environment (Docker container or local directory) where an agent executes code
- **Host** -- Your machine where Sandcastle runs and the real git repo lives
- **Agent** -- The AI coding tool invoked inside the sandbox (e.g. Claude Code)
- **Iteration** -- A single invocation of the agent, producing at most one commit

## Quick Start

### 1. Install

```bash
npm install --save-dev @ai-hero/sandcastle
```

### 2. Initialize

Run `npx @ai-hero/sandcastle init` in your repository to scaffold the config directory:

```bash
npx @ai-hero/sandcastle init
```

This creates a `.sandcastle/` directory with a Dockerfile, prompt file, and configuration.

### 3. Run

```bash
npx tsx .sandcastle/main.ts
```

Sandcastle will build the Docker image, sync your code into the sandbox, and start the agent.
